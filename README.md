Getting Started
Prerequisites

    Go 1.16 or higher
    GCP account with Cloud Storage API enabled
    PostgreSQL 10 or higher
    Redis 4 or higher

## Installing

Clone the repository

    git clone https://github.com/username/repo.git
    
Install dependencies
    
    go mod download
    
Set environment variables
    
    cp .env.example .env
    
Run the app
    
    make server
